Project Overview

Cairo2Capital is a comprehensive transportation management system that evolved from a simple map display into a sophisticated platform with full OOP implementation. The project showcases the journey from basic web development to advanced system architecture.

Foundation & Core Features

Restructured the entire codebase using Object-Oriented Programming principles. Added ticket pricing calculations, enhanced user interface, and improved system architecture.

Technical Implementation

HTML5 CSS3 JavaScript PHP MySQL

The Cairo2Capital project demonstrates a complete full-stack web application development cycle. The frontend utilizes modern HTML5 and CSS3 features for responsive design, while JavaScript handles dynamic user interactions and real-time calculations.

The backend is powered by PHP with MySQL database integration, following MVC architecture patterns. The second phase transformation to OOP principles showcased advanced software engineering concepts and code maintainability practices.
